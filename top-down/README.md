# Вариант
>Входные данные: строка которая может содержать буквы английского алфавита, а также величина сдвига. 
>Результат работы алгоритма: зашифрованная входная строка с помощью шифра Вижинера.

# Задание
Необходимо реализовать консольное приложение, которое имеет примитивный текстовый интерфейс и меню, состоящее как минимум из четырех пунктов:
- Ввод исходных данных, как вручную, так и сгенерированных случайным образом
- Выполнение алгоритма по заданию
- Вывод результата
- Завершение работы программы
Особые указания:
- Алгоритм не может быть выполнен без введенных данных
- Результат не может быть выведен без обработки введённых данных
- Вывод результата не может быть осуществлен без выполнения алгоритма
- При вводе новых данных результаты выполнения алгоритма "сбрасываются"
- Варианты заданий, где указан массив слов подразумевают, что разделителями для слов являются пробельные символы и знаки препинания.

# Ход выполнения
1. итерация1 - все основные функции в виде заглушек с pass или return "образец";
2. итерация2 - прописать основные функции, добавить вспомогательные в виде заглушек с pass или return "образец";
3. итерация3 - выполнение алгоритма по заданию.
